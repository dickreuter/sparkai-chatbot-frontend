Software Security

We provide organisations with a secure environment to develop tenders and bids, without the risk of sensitive data exposure to third parties.

Secure Infrastructure: mytender.io operates on Amazon's secure and reliable cloud infrastructure. AWS is trusted by governments and leading organisations worldwide for its robust security measures.

Encryption at Rest and In Transit: Data is encrypted at rest using AES-256 encryption, both within EBS volumes and snapshots. In transit, data is encrypted using TLS 1.2 or higher protocols.

Access Control: Granular access controls ensure only authorised users can access your bid data. You can define user permissions and roles to manage access levels within your team.

Compliance: AWS adheres to a wide range of industry compliance standards, including HIPAA, PCI DSS, and GDPR. This ensures your data is handled according to the most stringent security regulations.

Virtual Private Cloud (VPC): mytender.io utilises a unique Virtual Private Cloud (VPC) for each client. A VPC acts as a secure and isolated network environment for your bid data, further segregating it from other services and public traffic.

Secure Data Transfer: Data transfer with open-source intelligence or the internet is strictly limited. mytender.io prioritises secure methods for data exchange. This can include:

S3 Buckets with Private Access: Uploading and downloading data can occur through private S3 buckets, which restrict access only to authorised users and applications.

Secure File Transfer Protocols: Protocols like SCP or FTPS can be used for secure file transfer, ensuring encryption during data movement.
